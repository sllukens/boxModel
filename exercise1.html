Width, margins and padding.

In this exercise, practice width, margins and padding to see how they effect the div below. The div has the id "testDiv", so you can alter its settings by adding rules to the #testDiv section above. I've provided some starting values. The div has a red background, so you can see the total size of the div, as well as div content. I have provided a ruler, created with CSS, so you can see how your changes alter the size of the div. The ruler has lines every 10 pixels, notice how the total size of the div, and the size of the content, change as you alter different settings. Create a new file in atom in this directory, and use it to jot down what you discover when you try the following:
  1. Set some different percentage values for width, and try resizing your browser window and seeing what happens to the div.
  2. Set a px value for width and resize your browser window, what happens?
  3. See how margin and padding settings affect the div differently
  4. Set padding or margin in % and then resize the brower window. What happens?
  5. Try two-value settings for margin and padding and see what happens.
  6. Experiement with margin-top, margin-bottom, margin-right, and margin-left. Try the padding versions as well. See what kinds of effects you can acheive. Notice how you can move the div around its container and the content around the interior of the div.
  7. Try different font-sizes

You can use the firefox inspector to get a better sense of how the size of the div is changing as the settings change. Go to the firefox menu and go to Developer->Inspector and hover over the div to see the content area and padding.
